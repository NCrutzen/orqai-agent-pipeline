## What Was Built

**Task 1 — `<RowVerdictDot>` presentational primitive**

Single named-export client component at `web/app/(dashboard)/automations/[swarm]/_shell/components/row-verdict-dot.tsx`. Renders an 8px circle, absolute-positioned at `left:4, top:50%, translateY(-50%)`, with `pointerEvents:none` (underlying row click still fires) and `zIndex:0` (sits under the selected-row left border).

Color encoding (CONTEXT D-2):

| `verdict` prop | Visual                                         |
| -------------- | ---------------------------------------------- |
| `"confirm"`    | filled `var(--v7-lime)`                        |
| `"override"`   | filled `var(--v7-amber)`                       |
| `"unclear"`    | filled `var(--v7-brand-secondary)`             |
| `null`         | transparent + `1px dashed var(--v7-text-dim)`  |

DOM hooks: `data-testid="row-verdict-dot"`, `data-verdict={verdict ?? "none"}`, `aria-hidden="true"`. Zero state, zero effects, zero event handlers. Inline-style only (no Tailwind), matching the existing V7 `<StageBadge>` convention from `_shell/row-list.tsx`.
