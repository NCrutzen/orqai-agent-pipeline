## Deviations from Plan

None on contracts. One mechanical adjustment:

**[Rule 3 - Blocking] vi.mock factory TDZ on hoisted `inngestSend`**
- **Found during:** Task 2 first run (`labeling-flip-cron.predictor.test.ts` + `actions.predictor.test.ts`)
- **Issue:** vitest hoists `vi.mock(...)` factories to the top of the module; modules under test (`labeling-flip-cron.ts`, `actions.ts`) call `inngest.send` / `inngest.createFunction` at module-load time, triggering the factory before the top-level `const inngestSend = vi.fn()` was initialized → `ReferenceError: Cannot access 'inngestSend' before initialization`.
- **Fix:** Switched both files to `vi.hoisted(() => ({ inngestSend: vi.fn() }))`, which hoists the spy declaration alongside the factory.
- **Files modified:** `labeling-flip-cron.predictor.test.ts`, `actions.predictor.test.ts` (in-file, before commit).
- **Commit:** `e576374` (single Wave 0 commit).
