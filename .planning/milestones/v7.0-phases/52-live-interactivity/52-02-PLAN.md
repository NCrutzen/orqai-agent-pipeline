# Plan 52-02: Kanban Board with dnd-kit + Server Action + Realtime Sync

## Goal
Replace the `Kanban` placeholder with a 5-column drag-and-drop board reading from `useRealtimeTable("jobs")`. Drops persist via a `moveJob` server action with optimistic UI + sonner toast revert. Keyboard-accessible via dnd-kit's KeyboardSensor. Seed test data via Management API.

### Task 5: KanbanJobCard component
- `web/components/v7/kanban/kanban-job-card.tsx`:
  - Props `{ job: SwarmJob; isDragOverlay?: boolean }`
  - Calls `useSortable({ id: job.id, data: { stage: job.stage } })`
  - Glass-gradient bg per UI-SPEC; padding 14px; radius `--v7-radius-mini`
  - Header: title (Cabinet 700 15.5px) + description (line-clamp-2 muted 14px)
  - Tag row: priority pill + risk pill + up to 3 derived tags

    - Risk pill (warn variant) when `priority IN ('high','urgent')` or `tags` contains "sla"/"blocked"/"risk"
  - When dragging: dnd-kit applies transform via inline style; we add `cursor: grabbing` + opacity 0.95
  - Pass dnd-kit `attributes` + `listeners` onto the article element

### Task 7: KanbanBoard root component
- `web/components/v7/kanban/kanban-board.tsx`:
  - Props `{ swarmId: string }` (swarmId currently unused in render but reserved for telemetry)
  - Reads `{ rows: jobs } = useRealtimeTable("jobs")`
  - Reads `useSearchParams()` and applies `getFilterPredicate(params.get("filter"))`
  - Optimistic overlay: `useState<Map<string, SwarmJobStage>>(new Map())`
  - `displayedJobs = jobs.map(j => overlay.has(j.id) ? { ...j, stage: overlay.get(j.id)! } : j)` (memoized)
  - `filteredJobs = displayedJobs.filter(predicate)` (memoized)
  - Group by stage: `byStage: Record<SwarmJobStage, SwarmJob[]>` (sort within stage by `created_at desc`)
  - DndContext with sensors per `52-RESEARCH.md` section 2
  - `onDragStart` -> set `activeJob` state
  - `onDragEnd` -> compute `newStage`, short-circuit if same, apply optimistic, fire transition with `moveJob(jobId, newStage)`, on error revert + sonner toast
  - DragOverlay rendering `<KanbanJobCard job={activeJob} isDragOverlay />`
  - Reconcile overlay with realtime per `52-RESEARCH.md` section 6 (useEffect prune)
  - Outer GlassCard panel with header eyebrow ("Job board") + title ("Kanban for business stages, not micro-steps") + right meta `{filteredJobs.length} jobs total`
  - Inner grid `grid-cols-5` with the 5 columns

**Commit:** `feat(52-02): add KanbanBoard with dnd-kit + optimistic moveJob + Realtime sync`
