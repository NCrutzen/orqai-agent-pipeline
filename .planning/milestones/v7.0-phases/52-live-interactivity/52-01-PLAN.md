# Plan 52-01: Terminal Event Stream + Ring Buffer

## Goal
Replace the `Terminal` placeholder in `swarm-layout-shell.tsx` with a Claude-style live event stream that reads from `useRealtimeTable("events")`, holds at most 500 events in a per-swarm ring buffer, auto-scrolls when the user is at the bottom, and exposes Pause / Clear / "N new events" affordances.

### Task 2: Ring buffer store (`web/lib/v7/terminal/event-buffer.ts`)
- Implement `EventBufferStore` class per `52-RESEARCH.md` section 1, with:
  - capacity 500 (constructor arg, default 500)
  - `subscribe`, `getSnapshot`, `getServerSnapshot` methods
  - `pushMany(events: AgentEvent[])` with id-dedup + FIFO eviction
  - `setPaused(p: boolean)` with side-buffer flush on un-pause
  - `clear()`
  - `paused` boolean exposed read-only
- Module-scoped `Map<string, EventBufferStore>` keyed by `swarmId`; `getStore(swarmId)` factory
- Frozen `EMPTY_SNAPSHOT: AgentEvent[] = Object.freeze([]) as AgentEvent[]`
- Type imports from `@/lib/v7/types`
- No React imports here (pure store)

**Commit:** `feat(52-01): add per-swarm event buffer module store with FIFO eviction`

### Task 3: Hook (`web/lib/v7/terminal/use-event-buffer.ts`)
- `"use client"` module
- `useEventBuffer(swarmId: string)` calls `useSyncExternalStore(store.subscribe, store.getSnapshot, store.getServerSnapshot)`
- Returns `{ events, paused, setPaused, clear, pushMany }` -- the latter two delegate to the store
- `paused` exposed as a derived state via a tiny version subscription (or read directly from `store.paused` -- since `setPaused` calls `emit()`, components re-render and pick up the new value)

**Commit:** `feat(52-01): add useEventBuffer hook backed by useSyncExternalStore`

### Task 4: Format helpers (`web/lib/v7/terminal/format.ts`)
- `formatTime(iso: string): string` -- `HH:mm:ss` via `Intl.DateTimeFormat("en-GB", { hour, minute, second, hour12: false })`. Cache the formatter at module scope.
- `formatPayload(event: AgentEvent): string` -- builds a short payload string. Examples:
  - `tool_call`: `"{agent} -> tool: {tool}"` (tool from `content.tool` if present)
  - `tool_result`: `"{agent} <- tool: {tool}"`; if `content.error`, append ` (error: {error})`
  - `thinking`: `"{agent} reasoning: {span_name}"`
  - `done`: `"{agent} done: {span_name}"`
  - `waiting`: `"{agent} waiting: {span_name ?? 'human gate'}"`
  - `error`: `"{agent} error: {reason ?? 'unknown'}"`
  - `delegation`: `"{agent} delegated"`
- `EVENT_TYPE_LABELS: Record<AgentEventType, string>` per UI-SPEC chip-label table
- `EVENT_TYPE_CHIP: Record<AgentEventType, { bg: string; border: string; label: string }>` per UI-SPEC color map

**Commit:** `feat(52-01): add terminal event formatters and chip color map`

### Task 6: TerminalStream panel + bridge + auto-scroll + controls
- `web/components/v7/terminal/terminal-stream.tsx`:
  - Props: `{ swarmId: string }`
  - Reads `{ rows: realtimeEvents } = useRealtimeTable("events")`
  - `const { events, paused, setPaused, clear, pushMany } = useEventBuffer(swarmId)`
  - **Bridge effect:** `useEffect(() => pushMany(realtimeEvents), [realtimeEvents])` -- pushMany is idempotent so resubmitting all is safe; for performance, optionally diff by `id` first
  - **Reverse-chronological note:** `useRealtimeTable("events")` returns rows in `created_at desc` order. The terminal expects oldest-first display (newest at bottom). Sort the realtime feed to ascending before push: `useMemo(() => [...realtimeEvents].sort((a,b) => a.created_at.localeCompare(b.created_at)), [realtimeEvents])`
  - **Auto-scroll:** per `52-RESEARCH.md` section 5
  - **Header:** GlassCard wrapping; eyebrow row with pulse dot + "Live event stream", title "Latest events from the swarm", right side has Pause/Resume + Clear icon-buttons (`Pause`, `Play`, `Eraser` from lucide). Pause shows `--v7-amber-soft` bg when active.
  - **Body:** `<div ref={scrollerRef} className="v7-terminal-shell">` containing rendered `TerminalRow`s; mark the LAST row `isLatest`
  - **Empty state:** when `events.length === 0`, render a single dim row: `> Awaiting events...`
  - **N new events pill:** sticky bottom-right, only when `missedCount > 0 && !followBottom`

**Commit:** `feat(52-01): add TerminalStream panel with auto-scroll + pause + clear`
