   - Tool design for efficiency: self-contained descriptions, minimize back-and-forth

   - Just-in-time context retrieval: load context when needed, not upfront
   - Long-horizon task management: checkpoints, state summarization, memory stores
   For each pattern: 1-2 sentence actionable guidance for generated agent instructions

4. **Agent Composability Principles**:
   - Prefer simple patterns over frameworks
   - Each agent should be independently testable
   - Explicit data contracts between agents
