# Phase 82.5 Plan 04: Wire RowVerdictDot into RowList — Summary

**One-liner:** RowList now mounts `<RowVerdictDot>` as an absolute-positioned overlay in every row and consumes the threaded `feedbackMap`, delivering R3's visible row-strip verdict signal without touching the grid layout.

## What Changed

Single-file integration. `web/app/(dashboard)/automations/[swarm]/_shell/row-list.tsx`:

1. Added import: `import { RowVerdictDot } from "./components/row-verdict-dot";`
2. Removed the `void feedbackMap;` forward-compat suppression — the prop is now actively consumed.
3. Added `position: "relative"` to each row `<li>` inline style so the absolutely-positioned dot anchors to the row.
4. Mounted `<RowVerdictDot verdict={feedbackMap?.[r.id] ?? null} />` as the first JSX child of each `<li>`, before `<StageBadge>`.
