# Phase 82.1 Plan 03: Widen row-list pane + tighten sender column — Summary

CSS-only polish fixing the `/stage-1` subject truncation ("R..", "A..") by adding 260px of grid headroom and tightening the shared sender column primitive.

## What Changed

### Task 1 — Sender column tightened (D-04)
**File:** `web/app/(dashboard)/automations/[swarm]/_shell/row-list.tsx`
**Commit:** `84cf1fc`

The `<span data-testid="row-sender">` style was tightened:
- `minWidth: 160` → `minWidth: 120`
- `maxWidth: 220` → `maxWidth: 200`

All other style properties (fontSize 13, color, overflow hidden, textOverflow ellipsis, whiteSpace nowrap) unchanged. Subject span (`<span data-testid="row-subject">`) untouched — keeps `flex: 1` + ellipsis per D-06.

Because `_shell/row-list.tsx` is the shared primitive used by all stages (0, 1, 2, 3, 4), this change applies uniformly across the dashboard.

### Task 2 — Stage 1 grid widened (D-05)
**File:** `web/app/(dashboard)/automations/[swarm]/stage-1/client-shell.tsx`
**Commit:** `b0315d3`

The two-column grid template at the row-list/detail-pane container was widened:
- `gridTemplateColumns: "minmax(380px,460px) 1fr"` → `gridTemplateColumns: "minmax(540px, 720px) 1fr"`

Effect: subject column has visible room (~200px+) on a 1280px+ viewport. No other style on the container changed (gap, minHeight, child props all preserved).
