### Task 1 â€” actions.ts predictor lookup + insert

`verdictSchema` was extended with `email_id: z.string().min(1)`. Before the `agent_runs.insert`, `recordVerdict` performs:

```typescript
const { data: stage1Event } = await admin
  .from("pipeline_events")
  .select("decision_details")
  .eq("email_id", parsed.email_id)
  .eq("stage", 1)
  .eq("swarm_type", parsed.swarm_type)
```
