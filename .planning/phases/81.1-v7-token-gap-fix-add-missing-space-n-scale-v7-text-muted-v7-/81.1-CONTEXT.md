<domain>
## Phase Boundary

Pure CSS token gap fix. The `_shell/StageTabStrip`, `_shell/PageHeader`, and Phase 76 Stage 3 surfaces (`filter-chips`, `detail-pane`, `row-list`, `action-stack`, `inline-editor`, `reason-pill`, `conf-bar`) all reference `var(--space-N)`, `var(--v7-text-muted)`, and `var(--v7-border)` ‚Äî none of which exist in `web/app/globals.css`. Result: every `gap`, `padding`, and `margin` referencing the space scale resolves to invalid ‚Üí `0`, causing tabs and columns to collide with no separation.

**In scope:**
- Add the missing `--space-1` through `--space-7` scale to `:root, [data-theme="light"]` in `web/app/globals.css` (values from `.claude/skills/sketch-findings-agent-workforce/sources/themes/v7.css:63-69`).
- Add `--v7-text-muted` and `--v7-border` aliases referencing the existing `--v7-muted` and `--v7-line` tokens.
- Verify visual smoke on `/stage-0`, `/stage-1`, `/stage-2`, `/stage-3`, `/stage-4` ‚Äî tab strip + column gaps render correctly.

**Out of scope:**
- Renaming the 51 `--v7-text-muted` / `--v7-border` usages back to canonical names (aliases are cheaper than a sweep and stay forward-compatible).
- Any new tokens beyond what's already referenced by existing code.
- Stage 3 surface logic ‚Äî the column collisions are pure CSS, not layout-component bugs.

</domain>

<decisions>
## Implementation Decisions

- **D-01:** Use the v7 canonical scale verbatim ‚Äî `--space-1: 4px; --space-2: 8px; --space-3: 12px; --space-4: 16px; --space-5: 20px; --space-6: 24px; --space-7: 32px`. Source: Sketch v7.css lines 63-69.
- **D-02:** Place the tokens inside `:root, [data-theme="light"]` (line 173 closing brace). The space scale is theme-agnostic ‚Äî no overrides needed in `[data-theme="dark"]`. CSS inheritance handles both themes.
- **D-03:** Add aliases `--v7-text-muted: var(--v7-muted)` and `--v7-border: var(--v7-line)` rather than sweeping 51 usages. Cheap, preserves intent (the renamed tokens are semantically synonymous), no behavior change.
- **D-04:** No new tests. Existing `stage-tab-strip.test.tsx`, `page-shell.test.tsx`, and Stage 3 tests assert structural behavior, not pixel spacing. Manual smoke is the gate.

</decisions>
