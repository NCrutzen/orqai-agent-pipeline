# Plan 51-01: Subagent Fleet Cards

**Goal:** Replace the fleet placeholder in `swarm-layout-shell.tsx` with real cards reading from `swarm_agents` via the Phase 49 realtime context. Cards are clickable and open the Phase 51-03 drawer.

## Requirements covered
FLEET-01, FLEET-02, FLEET-03, FLEET-04

## Tasks

### T1. Zod schema for agent metrics/skills
Create `web/lib/v7/fleet/agent-metrics.ts`:
- Export `agentMetricsSchema` = `{ active_jobs, queue_depth, error_count }` with safe integer defaults (0)
- Export `agentSkillsSchema` = `z.array(z.string()).default([])`
- Export `parseAgentMetrics(raw)` + `parseAgentSkills(raw)` helpers returning typed objects with defensive fallbacks (never throw; bad JSONB -> zeros / empty)

### T2. `<AgentStateBadge>` component
Create `web/components/v7/fleet/agent-state-badge.tsx`:
- Input: `status: SwarmAgentStatus`
- Renders pill with 8px dot + copy per the UI-SPEC color/copy map
- `active` dot gets the pulse animation (CSS keyframes already in globals.css from Phase 48; if not, add a tiny local `@keyframes` via inline style -- no new token)

### T3. `<SubagentFleetCard>` component
Create `web/components/v7/fleet/subagent-fleet-card.tsx`:
- Props: `agent: SwarmAgent`, `onClick: () => void`
- Wraps `<GlassCard>` with className override `rounded-[var(--v7-radius-card)] cursor-pointer transition hover:translate-y-[-3px] hover:shadow-[var(--v7-glass-shadow-heavy)] focus-visible:outline-2 focus-visible:outline-[var(--v7-teal)]`
- Layout: top row (name Cabinet 20/700 left, AgentStateBadge right), role line (`--v7-muted` min-height 40px so cards align), metrics 3-col grid (Active / Queue / Errors), skill row (max 6 visible; overflow count "+N more")
- Accessibility: `role="button"`, `tabIndex={0}`, Enter/Space triggers `onClick`

### T4. `<SubagentFleet>` wrapper
Create `web/components/v7/fleet/subagent-fleet.tsx`:
- `"use client"`
- Consume `useRealtimeTable("agents")`
- Sort by `agent_name` ASC for deterministic order
- Empty state: "No subagents registered" heading + body copy per UI-SPEC, inside a `<GlassCard>` matching the placeholder footprint
- Non-empty: section header (eyebrow "Subagent fleet" + subtitle + "{N} specialists" badge) and `grid-cols-1 md:grid-cols-2 xl:grid-cols-4 gap-[14px]`
- Click handler: calls `useDrawer().openAgent(agent.agent_name)` (DrawerContext from Plan 51-03 -- import will resolve after 51-03 ships; to keep 51-01 independent, pass a prop `onAgentClick` that the shell wires up)

### T5. Wire into layout shell
Modify `web/components/v7/swarm-layout-shell.tsx`:
- Replace the subagent fleet `PlaceholderRegion` block with `<SubagentFleet onAgentClick={setOpenAgent} />`
- `setOpenAgent` comes from `useDrawer()` (added in 51-03) -- pre-wire the prop so fleet ships before drawer, but leave `onAgentClick={noop}` temporarily; 51-03 replaces the noop

### T6. Verify types + lint
- `cd web && npx tsc --noEmit` -- must pass (no new errors)
- `cd web && npx eslint components/v7/fleet` -- must pass
