# Plan 51-03: Agent Detail Drawer

**Goal:** Slide-out drawer opened by clicking a fleet card. Shows agent name, role, active count, avg cycle time, mini-hierarchy description, communication timeline (last 5 events grouped by trace_id), and a workflow tag row.

## Requirements covered
DRAW-01, DRAW-02, DRAW-03, DRAW-04

## Tasks

### T1. DrawerContext
Create `web/components/v7/drawer/drawer-context.tsx`:
- `"use client"`
- `DrawerContext` with `{ openAgent: string | null, setOpenAgent: (name: string | null) => void }`
- `<DrawerProvider>` wraps children
- `useDrawer()` hook throws if used outside provider
- Minimal -- just `useState` + context

### T2. Cycle time calculator
Create `web/lib/v7/drawer/cycle-time.ts`:
- `computeAverageCycleMs(events: AgentEvent[], agentName: string): number | null`
- Pairs `thinking` and `done` rows by `span_id`, filters to `agent_name === agentName`, computes duration = `ended_at - started_at` (use `done.ended_at - thinking.started_at`), bounds to last 20 completed spans
- Returns null if no complete pairs
- Format helper `formatCycle(ms)` returns "18m" / "2m 14s" / "45s" / "--"

### T3. Timeline grouper
Create `web/lib/v7/drawer/timeline.ts`:
- `groupEventsByTrace(events: AgentEvent[], agentName: string, limit = 5): Array<{ traceId: string | "unknown", events: AgentEvent[] }>`
- Filters to agent, sorts by `created_at` desc, slices top `limit`, groups by `content.trace_id` preserving first-seen order
- Each AgentEvent in output retains its `content` (JSONB) so UI can pull `span_name` / `tool`

### T4. `<DrawerTimeline>` component
Create `web/components/v7/drawer/drawer-timeline.tsx`:
- `"use client"`
- Props: `groups: ReturnType<typeof groupEventsByTrace>`, `statusColor: string` (from agent.status -> token)
- For each group: `<small>trace {id.slice(0,8)}</small>` header in `--v7-faint`, then rows of dot+text
- Each row: `grid-template-columns: auto 1fr; gap: 12px; align-items: start`
- Dot: 12px circle with box-shadow halo per UI-SPEC
- Text: "{event_type} -- {span_name || tool}" in `--v7-muted`, timestamp in `--v7-faint` monospace 11px
- Empty state copy: "No recent activity."

### T5. `<AgentDetailDrawer>` component
Create `web/components/v7/drawer/agent-detail-drawer.tsx`:
- `"use client"`
- No props -- reads `useDrawer()` for open state
- Reads `useRealtimeTable("agents")` to find the agent by `agent_name === openAgent`
- Reads `useRealtimeTable("events")` for timeline + cycle calculation
- Renders `<Sheet open={openAgent !== null} onOpenChange={(open) => !open && setOpenAgent(null)}>` wrapping `<SheetContent side="right" showCloseButton={false} className="v7-drawer-content">` with:
  - Drawer head: eyebrow "Recursive agent view" with pulse dot, Cabinet 24px agent name, Satoshi muted role, custom pill-shaped "Close" button top-right (`<SheetClose asChild>`)
  - Drawer body:
    - 2-col KPI grid (Active from metrics.active_jobs, Avg cycle from computeAverageCycleMs)
    - Mini-hierarchy panel (static text per UI-SPEC: `"${agent_name} behaves like a micro-orchestrator for ${role}: intake, run, escalate or delegate."`)
    - Timeline panel: heading "Recent communication" + count "Last 5 events" on the right, then `<DrawerTimeline>`
    - Workflow panel: tag row `["Intake", "Run", "Verify", "Escalate", "Done"]` with "Done" using `--v7-teal-soft` bg
- When `openAgent` is null OR no matching agent found, render nothing (`<Sheet open={false}>`)
- Add the `v7-drawer-content` Tailwind override in `globals.css` to apply 28px radius + 16px margin insets + glass bg (or inline via className)

### T6. Wire into shell
Modify `web/components/v7/swarm-layout-shell.tsx`:
- Wrap shell contents with `<DrawerProvider>`
- Mount `<AgentDetailDrawer />` as a sibling at the shell root (outside the scrollable main)
- Replace the temporary noop `onAgentClick` with a small wrapper that calls `useDrawer().setOpenAgent(agent.agent_name)`
- Since `useDrawer` is a hook and we're in a server-ish layout shell, extract the click wiring into a tiny client component `<FleetClickBinding>` that passes the callback to `<SubagentFleet>`

### T7. Verify + accessibility
- Keyboard: Tab cycles inside drawer; Esc closes; clicking backdrop closes
- Focus trap: Radix handles automatically
- Type check: `cd web && npx tsc --noEmit`
