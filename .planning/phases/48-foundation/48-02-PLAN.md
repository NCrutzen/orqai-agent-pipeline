2. Create `supabase/migrations/20260415_v7_foundation.sql` with all 4 tables. Use the exact schema from the RESEARCH.md code example:

**agent_events table:**
- id UUID PK, swarm_id UUID FK to projects(id) ON DELETE CASCADE, agent_name TEXT NOT NULL
- event_type TEXT NOT NULL with CHECK constraint: 'thinking', 'tool_call', 'tool_result', 'waiting', 'done', 'error', 'delegation'
- span_id TEXT, parent_span_id TEXT, content JSONB DEFAULT '{}'
- started_at TIMESTAMPTZ, ended_at TIMESTAMPTZ, created_at TIMESTAMPTZ DEFAULT NOW()
- Indexes: (swarm_id, created_at DESC), (span_id)

**swarm_jobs table:**
- id UUID PK, swarm_id UUID FK to projects(id) ON DELETE CASCADE, title TEXT NOT NULL, description TEXT
- stage TEXT NOT NULL DEFAULT 'backlog' with CHECK: 'backlog', 'ready', 'progress', 'review', 'done'
- priority TEXT DEFAULT 'normal' with CHECK: 'low', 'normal', 'high', 'urgent'
- assigned_agent TEXT, tags JSONB DEFAULT '[]', position INTEGER DEFAULT 0
- created_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW()
- Index: (swarm_id, stage)

**swarm_agents table:**
- id UUID PK, swarm_id UUID FK to projects(id) ON DELETE CASCADE, agent_name TEXT NOT NULL, role TEXT
- status TEXT DEFAULT 'idle' with CHECK: 'idle', 'active', 'waiting', 'error', 'offline'
- parent_agent TEXT, metrics JSONB DEFAULT '{}', skills JSONB DEFAULT '[]', orqai_deployment_id TEXT
- created_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW()
- UNIQUE(swarm_id, agent_name)
- Index: (swarm_id)

**swarm_briefings table:**
- id UUID PK, swarm_id UUID FK to projects(id) ON DELETE CASCADE, narrative TEXT NOT NULL
- metrics_snapshot JSONB DEFAULT '{}', generated_at TIMESTAMPTZ DEFAULT NOW()
- expires_at TIMESTAMPTZ DEFAULT (NOW() + INTERVAL '30 minutes')
- Index: (swarm_id, generated_at DESC)

**Realtime:**

**RLS:**

- SELECT policy for authenticated role (USING true) on all 4 tables
